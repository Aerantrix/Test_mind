  1. 🔴 Language switcher в авторизованной части (~03:30). Сейчас элемент либо отсутствует, либо абсолютно не находится. Если он реально в App Header — пометить иконкой 🌐, увеличить контраст, добавить tooltip. AI-ассистент даёт инструкцию, которая не приводит к успеху — это ломает доверие к Help.
  2. 🔴 Удалённый митинг доступен по той же ссылке без approval (~10:00). При delete митинга ссылка должна возвращать 404 или «meeting was cancelled».
  3. 🟠 Управление ролями в команде не находится (~12:30). Самый базовый team-management не найден за минуты.
  4. 🟠 Light/Dark переключатель вынести из Application Preferences наверх в Themes (~02:00).
  5. 🟡 Чёткое разделение «Display language» и «Translation language» (~01:00). Сейчас формулировка «Translation language, same as interface language» путает — это разные вещи.
  6. 🟡 Reset в Theme colors — либо чинить, либо убирать (~02:30).
  7. 🟡 «Create channel without contacts» — добавить empty-state CTA «Invite first contact» (~07:30).
