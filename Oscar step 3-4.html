Примечание Это та же сессия, что EK 3.mp4, записанная со стороны Oscar. Cross-tester verification ниже.

🔁 Подтверждённые находки (совпадают с EK 3)

  • 🔴 Кнопка смены языка интерфейса не находится в авторизованной части — Oscar тоже не нашёл, согласен с EK: «I can’t find any button to change the language on the application».
  • 🔴 Удалённый митинг остаётся доступен по той же ссылке без approval — Oscar подтвердил со своей стороны: «I joined it without permission» после того, как EK удалил митинг.
  • 🟠 Назначение admin role не находится — Oscar подтвердил: «We have no button to grant admin role to Oscar».
  • 🟠 Translation language vs display language — путаница: «Translation language, same as interface language». Понял через эксперимент: «This is so I can have display language be one, then I can set a default translation language».
  • 🟡 Theme — Light/Dark переключатель спрятан в Application Preferences.
  • 🟡 Create channel требует контакты — нет очевидной точки старта.

✅ Подтверждено как «работает»

  • Pre-Join Setup со стороны Oscar: «The link opened, it shows me camera setup with the calm music on the background. So I can choose a mic, it actually works».
  • Calming music на фоне Pre-Join — Oscar тоже отметил.
  • Speaker test работает: «На спикер-тест, мою музыку стало очень громко».
  • Approval flow (host видит pop-up «wants to join») работает корректно.

📋 Что не покрыто

То же, что у EK 3: - Массовое приглашение. - Полный roles flow. - Удаление участника, выход из команды. - 17 языков интерфейса — не пробовали.

🚦 Приоритеты

Идентичны EK 3, теперь с подтверждением: 1. 🔴 Language switcher в авторизованной части — оба тестировщика не нашли. 2. 🔴 Удалённый митинг доступен — воспроизведено с обеих сторон. 3. 🟠 Roles management — не находится.
