🚦 Priorities

  1. 🔴 Documents are accessible via direct link without login — serious security/privacy bug. If documents are team-internal — this is a leak. Right now the link works as "public, anyone with link".
  2. 🟠 Domain Management missing from Integrations — gap between what was promised and reality.
  3. 🟠 Google Drive / OneDrive missing — same thing.
  4. 🟠 AI editor completion not found / missing.
  5. 🟠 Empty message on copy/paste of a link — rendering broken on both sides.
  6. 🟠 XML error on revoke (confirmation of EK 11-16).
  7. 🟡 Live chat support — clarify plan: is it for everyone or only Business+? Right now the testers aren't sure.
  8. 🟡 Standalone Translate (/translate) — add an explicit UI entry point so it's not confused with the chat feature.
