🚦 Рекомендации команде

  1. Срочно — топ-10 критических багов выше (особенно recording, document access, удалённый митинг).
  2. Спринт — medium-баги (XML-ошибка, document blank, language switcher).
  3. Тестировщикам — 3 запросные сессии: §10 (Запись), §13 (Безопасность, нужен Business+), §15 (Mobile полноценно).
  4. В инструкции для тестировщиков — вычеркнуть «восстановление пароля» (паролей нет); проговорить, что для §13–14 нужен Business+.
